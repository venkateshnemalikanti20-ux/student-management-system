# Student Management System

A full-stack Student Management System built using Node.js, Express.js, MySQL, HTML, CSS, and JavaScript.

## Features

* Add Student
* View Students
* Update Student Details
* Delete Student
* MySQL Database Integration
* Cloud Database Connectivity using Aiven

## Technologies Used

* Node.js
* Express.js
* MySQL
* HTML
* CSS
* JavaScript

## CRUD Operations

* Create Student
* Read Students
* Update Student
* Delete Student
